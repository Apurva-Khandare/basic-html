# The basic info of HTML
The full form HTML is hyper text markup language thats simple and easy to use 
their are various tags in html that gives webpage a different colour or font size
## THE FOLLOWING IS THE BASIC CODE OF HTML:
``` html 
<!doctype html>
 <html>
    <head>
        <title>title of the project</title>
    </head>
<body>
    body of html code
</body>
 </html>
 ```

 ### The html used following basic tags:
 - `<b></b>` to give bold effect to the text
 - `<u></u>` to give underline to the text
 - `<i></i>` to give italic font to the text

 ### Their are two types of tags:
 1)paired tags
 2)container tags
